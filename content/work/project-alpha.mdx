# Project Alpha

A mobile-first experience that reimagines how users complete core tasks on the go.

## Challenge

The existing flow required too many steps. Users abandoned tasks midway.

## Solution

We redesigned the flow around a single primary action, with secondary options discoverable but not intrusive.

## Outcome

Task completion rate increased by 40%. User satisfaction scores improved significantly.
