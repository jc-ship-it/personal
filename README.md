# zhangjiachang.com

Personal website for Zhang Jiachang — Interaction Designer.  
Portfolio, blog, and professional presence. Target: zhangjiachang.com on Vercel.

## Stack

- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Content**: MDX (Phase 2)
- **Deploy**: Vercel

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Project Structure

```
/app           # Routes & pages
/components    # Reusable UI
/content       # Blog & Work MDX (Phase 2)
/lib           # Utilities
/public        # Static assets
```

## Phases

1. **Phase 1**: Home, About, Contact, navigation, dark/light mode, responsive
2. **Phase 2**: Blog & Work (MDX), content rendering
3. **Phase 3**: Admin panel (GitHub API, image upload)
4. **Phase 4**: Polish (SEO, OG, RSS, animations)

## Deploy

Push to GitHub → Import in [Vercel](https://vercel.com) → Bind `zhangjiachang.com`.
